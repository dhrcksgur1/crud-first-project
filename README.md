## 도메인 별 서비스 기능 명세
### 1. 게시판(Board) 기능
> - 게시판에 대한 생성, 조회, 수정, 삭제

### 2. 게시글(Post) 기능
> - 게시글에 대한 생성, 조회, 수정, 삭제
> - **키워드 검색** 으로 원하는 게시글 검색
> - **페이지네이션** 적용으로 성능 향상 및 사용자 편의성 개선

### 3.  댓글(Comment) 기능
> - 댓글에 대한 생성, 조회, 수정, 삭제
> - **페이지네이션** 적용으로 성능 향상 및 사용자 편의성 개선
